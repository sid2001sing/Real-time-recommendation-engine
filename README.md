Dynamic AI-Powered Recommendation Engine

A sophisticated real-time recommendation system that adapts to user search queries, powered by MongoDB and AI-driven content analysis.

## Features

-  Enter any search query and get personalized recommendations
- ** Query-specific trending analysis with live URLs
- **Intent analysis and semantic similarity scoring
- **Scalable database with aggregation pipelines
- **All recommendations include actual URLs from verified sources
- ** Instant adaptation to search queries



## How to Use

### 1. **Dynamic Search**
- Enter any search query (e.g., "python programming", "best movies", "fitness tips")
- Watch both recommendation sections adapt in real-time
- Get autocomplete suggestions as you type

### 2. **AI-Powered Recommendations**
- Click "AI Search-Powered" to get query-based recommendations
- Each recommendation includes relevance scores and real URLs
- Sources include Python.org, IMDb, Amazon, Coursera, etc.

### 3. **Real-Time Trending**
- Trending content automatically adapts to your search query
- Get trending news, discussions, and updates for your topic
- Direct links to Google Trends, Reddit, Twitter, and news sources
